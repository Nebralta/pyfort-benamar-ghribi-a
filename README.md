Fort Boyard Simulator Projet EFREI TI101 – Programmation en Python Groupe A

Présentation Générale Contributeurs Adam GHRIBI Aziz BENAMAR Description Dans le cadre d’un projet scolaire en programmation Python, nous avons conçu un mini-jeu inspiré de Fort Boyard. L’objectif principal est de réussir différentes épreuves pour collecter des clés et accéder à la salle du trésor.
Fonctionnalités Principales Épreuves variées : énigmes mathématiques, défis de logique et jeux de hasard. Gestion du déroulement du jeu : création d’équipes, choix des épreuves, comptabilisation des résultats. Technologies Utilisées Langage de programmation : Python Bibliothèques : time, random, json, math Outils : PyCharm, GitHub

Principe du jeu : Création d’une équipe de maximum 3 joueurs, en saisissant leur nom, profession et rôle. Choix de l’épreuve à réaliser par chaque joueur. Gestion des résultats : comptabilisation des clés gagnées et évaluation de la victoire ou de la défaite. Structure des fonctions Le projet est organisé en trois groupes de fonctions :

Fonctions de jeu : Chaque épreuve est gérée par une fonction qui renvoie un booléen (True si le joueur gagne, False sinon). Fonctions utilitaires : Gèrent la création d’équipe et le menu des épreuves. Fonction principale (jeu()) : Coordonne l’ensemble du jeu et affiche le résultat final. Gestion des entrées et des erreurs Validation stricte des choix pour éviter les erreurs (par exemple, un menu refuse des options non disponibles). 3.
